# 🐾 Sistema de Clínica Veterinaria

Aplicación web desarrollada en **Java**, con interfaz en **HTML y CSS**, diseñada para la gestión integral de una clínica veterinaria. Permite administrar clientes, mascotas, citas médicas y registros básicos.

---

## 📋 Descripción

Este sistema permite a una clínica veterinaria llevar un control organizado de la información de sus clientes (dueños) y sus mascotas, facilitando la administración y mejorando la atención.

---

## 🎯 Objetivo

Desarrollar una aplicación que permita:

* Registrar clientes y mascotas
* Gestionar citas veterinarias
* Consultar información de manera rápida
* Mantener un control eficiente de los datos

---

## 🚀 Características

* 👤 Gestión de clientes (CRUD)
* 🐶 Registro de mascotas
* 📅 Control de citas
* 🔍 Búsqueda de información
* ✏️ Edición y actualización de datos
* ❌ Eliminación de registros

---

## 🛠️ Tecnologías Utilizadas

* ☕ Java

* 🗄️ Base de datos (MySQL o SQLite)
